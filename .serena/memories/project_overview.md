ScriptVault is a Flutter macOS app for saving, editing, and running shell scripts. Application code is organized by layer: lib/data for repositories and services such as script storage/execution, lib/domain for script models and run result types, lib/ui for screens/view models/widgets, lib/router for routing, macos for the native runner and entitlements, and test for Flutter unit/widget tests. Generated build output lives in build/ and should not be edited directly.
