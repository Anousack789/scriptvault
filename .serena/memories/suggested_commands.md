Use FVM for Flutter and Dart commands.
- fvm flutter pub get: install dependencies.
- fvm flutter run -d macos: run the macOS app locally.
- fvm flutter test: run all tests.
- fvm flutter analyze: run static analysis with flutter_lints.
- fvm dart format lib test: format Dart source and tests.
- fvm flutter build macos: create a macOS release build.
Useful Darwin shell commands: git status, rg, rg --files, ls, sed -n 'start,endp' file.
